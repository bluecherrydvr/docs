FACTORY_DEFAULT

Description

Restore the configuration of device to the factory default settings. It will reboot the device after the configuration saved into flash automatically.

Input Argument

None

Attribute

  • Not On-Fly

Return of Message

Always return OK

Applicability

  • WPL Platform : All Devices
  • EPL Platform : All Devices
  • PlatfotmT : All Devices
  • Multi-Channel Encoder : All Devices

Status

Valid

See Also

SAVE_REBOOT, SAVE, REBOOT, CONFIG_RESET

Example

http://ip:port/cgi-bin/system?USER=Admin&PWD=123456&FACTORY_DEFAULT
or
http://ip:port/cgi-bin/cmd/system?FACTORY_DEFAULT
