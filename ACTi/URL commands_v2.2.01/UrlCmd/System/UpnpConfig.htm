UPNP_CONFIG

 

Description

Get or Set UPnP Configurations.

When the UPnP function is enabled in the device, the remote hosts which are UPnP enabled devices like Windows XP could discover the device automatically. Therefore, the hosts could visit the device's WEB page directly without knowing device's IP addresses.

Input Argument

state,name
where
state: 0 (disable) or 1 (enabled).
name: UPnP friendly name in 31 bytes character string. These characters should be readable and the space ,' ', character is acceptable. If state = 0, this argument could be omitted.
if state = 1 and the name argument is missing, the device will use its current UPnP Friend name setting to start its UPnP.

Attribute

  • Readable and Writable
  • On-Fly

Return of Message

  • Read Operation
UPNP_CONFIG='0'
or
UPNP_CONFIG='1,IPCAM-00001'
  • Write Operation
OK: UPNP_CONFIG='0'
or
OK: UPNP_CONFIG='1,IPCAM-00001'
or
ERROR: UPNP_CONFIG='0'

Applicability

  • Platform W: All encoder devices
  • Platform A: All devices
  • Platform T: All devices

Status

Valid

Example

  • Read Operation
http://ip:port/cgi-bin/system?USER=admin&PWD=123456&UPNP_CONFIG
or
http://ip:port/cgi-bin/cmd/system?UPNP_CONFIG
  • Write Operation
http://ip:port/cgi-bin/system?USER=admin&PWD=123456&UPNP_CONFIG=1,IPCAM-00001
or
http://ip:port/cgi-bin/cmd/system?UPNP_CONFIG=1,IPCAM-00001
