OEM_INFO

 

Description

Get the device OEM configurations.
There are three configurations will be returned in the NB devices. They are

  • OEM_SERIAL_NUM
  • OEM_MODEL_NAME
  • OEM_MODEL_DESC

These three configurations are NULL by default and could be set by the NB firmware maker tool. If they are NULL or the device is AC device, the return value of them are

OEM_SERIAL_NUM=''
OEM_MODEL_NAME=''
OEM_MODEL_DESC=''

In multiple channel video servers, the CHANNEL command with this URL is ignored. The OEM configurations in the system will be returnned.

Input Argument

None

Attribute

  • Read Only
  • Not On-Fly

Return of Message

  • AC device or NULL OEM Configurations

OEM_SERIAL_NUM=''
OEM_MODEL_NAME=''
OEM_MODEL_DESC=''

  • Not support this URL

ERROR: OEM_INFO not found

  • NB device and the OEM configurations are valid

OEM_SERIAL_NUM='XXX-123434'
OEM_MODEL_NAME='MM-CC'
OEM_MODEL_DESC='This is a camera'

Applicability

  • PlatformW : 1CH encoder (2.10.04)
  • PlatformA : All devices. (Encoder: 3.12.06, Decoder:3.04.00, QUAD: 3.04.01)
  • PlatformT : All Devices (Encoder: 4.07.11)
  • Multi-Channel Encoder : All Devices (Router: 2.07.01)

Status

Valid

See Also

None

Example

http://ip:port/cgi-bin/system?USER=Admin&PWD=123456&OEM_INFO
or
http://ip:port/cgi-bin/cmd/system?OEM_INFO
