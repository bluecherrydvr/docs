DECODER CGI Program

 

 

This CGI provides the URL command to access the Decoder devices.

Only root could set the configuration through this CGI. If the logon user is not root, all write commands will be treated as read commands.

NOTE:

The URL CGI which is old fashion CGI used for Decoder Devices. It will be divided into two parts of URLs. One will be migrated to the SYSTEM CGI and the other one will be moved and modified in this CGI.

This CGI is available only for the firmware version after B1X-A2-B2.05.06 in Platform W Decoders.
This CGI is available only for the firmware version after B1D-220-V3.02.00 in Platfomr A Decoders.
