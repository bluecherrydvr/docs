ZOOM_POSITION

Description

Get the current ZOOM Position in hexadecimal value.

Different Zoom module with different control value, for MT54G15, the control talble is

Zoom Heximal Decimal
1x 0x111 273
2x 0x223 547
3x 0x334 820
4x 0x446 1094
5x 0x557 1367
6x 0x669 1641
7x 0x780 1920
8x 0x892 2194
9x 0x1003 4099
10x 0x1115 4373

If the CHANNEL=n was not set with this URL to the Multi-Channel device, the CHANNEL 1 video server's ZOOM_POSITION configuration will be returned. If the CHANNEL 1 video server does not connect to the router, the error message will be returned.

Input Argument

None

Attribute

  • Read Only
  • On-Fly

Return of Message

  • Read Operation
ZOOM_POSITION='0xn'
where n is in hexadecimal for current zoom position.

Applicability

  • WPL Platform : All Encoders
  • EPL Platform : All Encoders
  • PlatfotmT : All Encoders
  • Multi-Channel Encoder : 4CH and 16CH Video Servers

Status

Valid

See Also

ADDR, PTZ_VENDOR, ZOOM, FOCUS, FOCUS_POSITION, MOVE, MOVE_LIMIT, MOVE_LIMIT_GET, MOVE_MAX_SPEED_GET, POSITION, XYZ_POS_GET

Example

  • Read Operation
http://ip:port/cgi-bin/encoder?USER=Admin&PWD=123456&ZOOM_POSITION
or
http://ip:port/cgi-bin/cmd/encoder?ZOOM_POSITION
