Encoder CGI Program

 

 

Only root could set the configuration through URLs in this CGI. If the logon user is not root, all write commands will be treated as read commands. The CGI is only available in the Encoder Devices. The URL in this CGI will be forwarded to the specific channel in the Multi-Channel Encoder platform if there is a CHANNEL commmand.

NOTE:

This CGI is new CGI for replacing mpeg4 CGI.
