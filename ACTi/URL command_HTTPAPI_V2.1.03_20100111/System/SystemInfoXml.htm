SYSTEM_INFO_XML

Description

Get the device system information in XML form. The system information is varied with different devices. Detail information , please check device capability design document.

Input Argument

None

Attribute

  • Read Only
  • Not On-Fly

Return of Message

The return of message is a xml data structure.
