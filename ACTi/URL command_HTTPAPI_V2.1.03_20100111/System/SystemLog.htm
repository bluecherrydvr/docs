SYSTEM_LOG

Description

Get the device system LOG messages. The system information is varied with different devices.

The version of the bootloader and device capability should be added in the system log. If the bootloader version does not exported to the kernel, the "UNKNOWN" will be shown. If there is no device capablity in the device, the "NONE" will be shown. The version field used in the devcap is DevConf.FwDevCap.head.ver.

The format of showing bootloader version is

Bootloader Version nnn
where nnn is the bootloader version.

The format of showing devcap version is

Devcap Version nnn
where nnn is the bootloader version. It should be shown in 4-digit hexidecimal value with 0x prefixor "NONE". For example, if nnn = 0x0001, the devcap version in system log should be Devcap Version 0x0001.

Input Argument

None

Attribute

  • Read Only
  • Not On-Fly
