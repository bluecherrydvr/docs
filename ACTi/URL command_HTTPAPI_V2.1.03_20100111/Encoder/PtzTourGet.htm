PTZ_TOUR_GET

Description

Get all valid PTZ tour entries configurations of a specific or current PTZ tour group.

If the device is not in the touring mode and there is no input argument, the settings of tour group 1 will be returned.

If the CHANNEL=n was not set with this URL to the Multi-Channel device, the error message will be returned.

The format of return is listed below.

PTZ_TOUR_GET='m,n,PresetPoint,Dwell,SpeedPan,SpeedTilt,SpeedZoom'

where,

  1. m: Tour Group ID. It started from 1 to 10.
  2. n: Entry index in the PTZ tour group. It started from 1 to 32 (EPL), 1 to 64 (PlatformT).
  3. PresetPoint: Entry of PTZ preset table associated with this PTZ tour entry. It is from 1 to 32 (EPL), 1 to 64 (PlatformT).
  4. Dwell: The timer for moving to and staying this entry in second. It is in the range between 1 and 255.
  5. Sequence: not used and fixed to 1
  6. SpeedPan: the speed in pan to move to the preset point. It should be in the range between 1 and 5.
  7. SpeedTilt: the speed in tilt to move to the preset point. It should be in the range between 1 and 5.
  8. SpeedZoom: the speed in zoom to move to the preset point. It should be in the range between 2 and 7.

The SpeedPan, SpeedTilt and SpeedZoom are shown in the PlatformT PTZ/Speed Dome only. However, these three arguments will not been show in EPL PTZ camera without any error.

If there is no valid entry, the 'm,0' will be returned.

Input Argument

None (the valid entries in current tour group will be returned)
or
m where Tour Group ID from 1 to 10
or
ERROR: invalid parameters

Attribute

  • Readable and Writable
  • On-Fly

Return of Message

PTZ_TOUR_GET='m,1,PresetPoint,Dwell,Sequence,SpeedPan,SpeedTilt,SpeedZoom'
PTZ_TOUR_GET='m,2,PresetPoint,Dwell,Sequence,SpeedPan,SpeedTilt,SpeedZoom'
PTZ_TOUR_GET='m,3,PresetPoint,Dwell,Sequence,SpeedPan,SpeedTilt,SpeedZoom'
.....
PTZ_TOUR_GET='m,31,PresetPoint,Dwell,Sequence,SpeedPan,SpeedTilt,SpeedZoom'
PTZ_TOUR_GET='m,32,PresetPoint,Dwell,Sequence,SpeedPan,SpeedTilt,SpeedZoom'
or
PTZ_TOUR_GET='m,0'

Applicability

  • WPL Platform : All Encoders
  • EPL Platform : All Encoders
  • PlatfotmT : All Encoders
  • Multi-Channel Encoder : 4CH and 16CH Video Servers

Status

Valid
