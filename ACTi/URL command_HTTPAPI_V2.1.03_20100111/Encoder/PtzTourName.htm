PTZ_TOUR_NAME

Description

Set or the name of a specific PTZ tour group or read the name of current tour group

If the device is not in the touring mode, the settings of tour group 1 will be returned when read.

In PTZ camera, there is 10 PTZ tour groups. The maximum length of tour group name is 31 characters. If the input string is longer than 31 characters, the input argument will be truncated.

If the CHANNEL=n was not set with this URL to the Multi-Channel device, the error message will be returned.

Input Argument

n,name
where
n is the tour group ID. 0: current working tour group. If the device is not in the touring mode, the settings of tour group 1 will be the target tour group to set the name. 1~10: tour group ID.
name is the character string (<32 bytes) for the tour group name
or
None (read name of current tour group)

Attribute

  • Readable or Writable
  • On-Fly

Return of Message

  • Write Operation

OK: PTZ_TOUR_NAME='n,name'
where n is the group ID and name is the tour group name of the tour group.
or
ERROR: invalid parameters

  • Read Operation

PTZ_TOUR_NAME='n,name'
where n is the current group ID and name is the tour group name of the tour group.

Applicability

  • WPL Platform : All Encoders
  • EPL Platform : All Encoders
  • PlatfotmT : All Encoders
  • Multi-Channel Encoder : 4CH and 16CH Video Servers

Status

Valid

See Also

PTZ_PRESET_SET, PTZ_PRESET_GET, PTZ_PRESET_GO, PTZ_TOUR_SET, PTZ_TOUR_GET, PTZ_TOUR_STATE, PTZ_TOUR_SCAN, PTZ_PARK_CONFIG

Example

  • Write Operation
http://ip:port/cgi-bin/encoder?USER=Admin&PWD=123456&PTZ_TOUR_NAME=1,Tour Group 1
http://ip:port/cgi-bin/cmd/encoder?PTZ_TOUR_NAME=1,Tour Group 1
  • Read Operation
    http://ip:port/cgi-bin/encoder?USER=Admin&PWD=123456&PTZ_TOUR_NAME
    http://ip:port/cgi-bin/cmd/encoder?PTZ_TOUR_NAME
