PTZ_PRESET_SET

Description

Set the PTZ preset entry configuration.

If the CHANNEL=n was not set with this URL to the Multi-Channel device, the error message will be returned.

PTZ_PRESET_SET Command Description
Entry,State,PanPos,TiltPos,ZoonPos,PanSpeed,TiltSpeed,ZoomSpeed,Dwell,Name Set a valid entry (State=1). Clear an entry (State=0)
Entry,1 Set a valid entry. The current speed and position of the pan, tilt and zoom will be used for this preset point
Entry,0 Clear a entry
  1. Entry: For EPL, Entry index in the PTZ Preset Table which has 32 entries. It started from 1 to 32.
    For PlatformT, Entry index in the PTZ Preset Table which has 64 entries. It started from 1 to 64
  2. State: Set the entry to be valid or invalid. It is 1 for a valid entry and 0 for an invalid entry.
  3. PanPos: absolute position in PAN.
  4. TiltPos: absolute position in TILT.
  5. ZoomPos: absolute position in ZOOM.
  6. PanSpeed: Speed of PAN movement, it is in the range of 1 and 5. The higher value, the faster movement.
  7. TiltSpeed: Speed of TILT movement, it is in the range of 1 and 5. The higher value, the faster movement.
  8. ZoomSpeed: Speed of ZOOM movement, it is in the range of 2 and 7. The higher value, the faster movement.
  9. Dwell: The timer for staying this entry in second. This configuration is not used now. Keep this setting to 10 when you set an entry.
  10. Name: name of this entry. The maximum length of this configuration is 31. The name will be truncated if it is longer than 31.

The PositionPan and PositionTilt are signed intergers. The PTZ camera might not have such wide range of these positions. However, the device will NOT validate them before parser and handle this URL. Please refer to the device specification to have correct

To clear the preset entry, this URL could be simplified to PTZ_PRESET_SET=Entry,0 where Entry is from 1. However, the complete arguments in this URL for clearing the entry is acceptable.

Input Argument

Refer to above command table

Attribute

  • Write Only
  • On-Fly

Return of Message

  • Write Operation
OK: PTZ_PRESET_SET='Entry,State,PanPos,TiltPos,ZoonPos,PanSpeed,TiltSpeed,ZoomSpeed,Dwell,Name'
or
 OK: PTZ_PRESET_SET='Entry,n'
or
ERROR: invalid parameters

Applicability

  • WPL Platform : All Encoders
  • EPL Platform : All Encoders
  • PlatfotmT : All Encoders
  • Multi-Channel Encoder : 4CH and 16CH Video Servers

Status

Valid

or

http://ip:port/cgi-bin/cmd/encoder?PTZ_PRESET_SET=1,0
