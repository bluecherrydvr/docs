HTTP_PROXY

Description

Get or Set the configuration for HTTP proxy server. The proxy server setting must go with HTTP_SERVER setting. HTTP server 1 goes with HTTP proxy 1, and HTTP server 2 goes with HTTP proxy 2.

Input Argument

        n,state,name,password,ip_addr,port
n:
     1: HTTP proxy 1
     2: HTTP proxy 2

state:
         1: Valid
         0: Invalid

name:  Maximum length is 32 bytes

password:    Maximum length is 32 bytes

ip_addr:    Maximum length is 64 bytes

port: 1~65535, default is 8080                      

Attribute

  • Readable and Writable
  • On-Fly

Return of Message

  • Read Operation

HTTP_PROXY='1,acti,acti,192.168.0.100,8080'
HTTP_PROXY='2,acti,acti,192.168.0.100,8080'

  • Write Operation
OK:HTTP_PROXY='1,acti,acti,192.168.0.100,8080'
or
ERROR:HTTP_PROXY='1,acti,acti,192.168.0.100,8080'

Applicability

  • EPL Platform : All Devices
  • PlatfotmT : All Devices
  • Multi-Channel Encoder : All Devices

Status

Valid

See Also

HTTP_SERVER

Example       

        read operation

          http://ip:port/cgi-bin/encoder?USER=Admin&PWD=123456&HTTP_PROXY
          or
          http://ip:port/cgi-bin/cmd/encoder?HTTP_PROXY

        write operation

http://ip:port/cgi-bin/encoder?USER=Admin&PWD=123456&HTTP_PROXY=1,1,acti,acti,192.168.0.100,8080
or
http://ip:port/cgi-bin/cmd/encoder?HTTP_PROXY=1,1,acti,acti,192.168.0.100,8080
