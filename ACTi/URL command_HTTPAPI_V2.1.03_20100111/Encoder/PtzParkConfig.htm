PTZ_PARK_CONFIG

Description

Enable or Disable PTZ Parking Operation.

The PTZ camera is stopped scanning as a movement in Pan or Tilt was issued. The camera could resume the scan function when there is no other action in a certain period of time, so-called park time,  if the PTZ Parking function is enabled.

If the CHANNEL=n was not set with this URL to the Multi-Channel device, the error message will be returned.

Input Argument

n,m where
n: 0: Disable, 1: Enable
m: Park time in 1~255 seconds. It could be ignored if n=0.

Attribute

  • Readable and Writable
  • On-Fly

Return of Message

  • Read Operation
PTZ_PARK_CONFIG='0,10'
  • Write Operation
OK: PTZ_PARK_CONFIG='0,10'
or
ERROR: PTZ_PARK_CONFIG='0,10'

Applicability

  • WPL Platform : All Encoders
  • EPL Platform : All Encoders
  • PlatfotmT : All Encoders
  • Multi-Channel Encoder : 4CH and 16CH Video Servers

Status

Valid

See Also

PTZ_PRESET_SET, PTZ_PRESET_GET, PTZ_PRESET_GO, PTZ_TOUR_SET, PTZ_TOUR_GET, PTZ_TOUR_NAME, PTZ_TOUR_SCAN

Example

  • Read Operation

http://ip:port/cgi-bin/encoder?USER=Admin&PWD=123456&PTZ_PARK_CONFIG
or
http://ip:port/cgi-bin/cmd/encoder?PTZ_PARK_CONFIG

  • Write Operation
http://ip:port/cgi-bin/encoder?USER=Admin&PWD=123456&PTZ_PARK_CONFIG='1,10'
or
http://ip:port/cgi-bin/cmd/encoder?PTZ_PARK_CONFIG='1,10'
