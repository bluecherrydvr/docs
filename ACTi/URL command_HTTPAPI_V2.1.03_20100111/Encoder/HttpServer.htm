HTTP_SERVER

Description

Get or Set the HTTP servers configurations.

When set, the specific HTTP server has to be addressed in the argument of this URL. Whe read, the configurations of two HTTP servers will be returned.

There are two HTTP server database, named HTTPD1 and HTTPD2, in the device. The notification message, image and video could be uploaded to these servers as events were triggered. The event has to be associated with one of these servers before it was enabled.

The Basic authentication is only supported here..If the user name is empty, the device will not do any authentication when it connects to the HTTP server.

Input Argument

        id,state,addr,port,name,password,timer

id: the index of the HTTP servers. 1: HTTPD1 and 2:HTTPD2

state: state of the HTTP server. 0:Disable, 1:Enable

addr: address of the HTTP server. The maximum length is 63.

port: the HTTP server port number. The default number

name: the user name to login the HTTP server. The maximum length is 31.

password: the user password to login the HTTP server. The maximum length is 31.

timer: the maximum time in seconds to establish a HTTP connection. It could be from 0~60. If it is 0, the connection timer is wrong.

If the length of the input argument is longer than its maximum length limitation, the error message will be returned.

Attribute

  • Readable and Writable
  • On-Fly

Return of Message

  • Read Operation

HTTP_SERVER='1,0,www.server1.com,80,myname,mypwd,10'
HTTP_SERVER='2,0,www.server2.com,80,myname,mypwd,10'

  • Write Operation
OK: HTTP_SERVER='1,0,www.server1.com,80,myname,mypwd,10'
or
 ERROR: invalid parameters

Applicability

  • EPL Platform : All Devices
  • PlatfotmT : All Devices
  • Multi-Channel Encoder : All Devices

Status

Valid

See Also

EVENT_RSPMSGn, EVENT_RSPIMGn, EVENT_RSPCMDn, EVENT_CONFIG

Example

  • read operation
http://ip:port/cgi-bin/encoder?USER=Admin&PWD=123456&HTTP_SERVER
or
http://ip:port/cgi-bin/cmd/encoder?HTTP_SERVER
  • write operation
http://ip:port/cgi-bin/encoder?USER=Admin&PWD=123456&HTTP_SERVER=1,1,www.server1.com,80,myname,mypwd,10
or
http://ip:port/cgi-bin/cmd/encoder?USER=Admin&PWD=123456&HTTP_SERVER=1,1,www.server1.com,80,myname,mypwd,10
