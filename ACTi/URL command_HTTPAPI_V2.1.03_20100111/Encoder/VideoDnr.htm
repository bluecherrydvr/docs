VIDEO_DNR

Description

Enable or disable the DNR function.

This URL is only available in some PlatformT encoders which has the VX1129 video processor. The DNR stands for Digital Noise Reduction and could reduce the noise and fixed the false color automatically.

Input Argument

0: Disabled, 1:Enabled.

Attribute

  • Readable and Writable
  • On-Fly

Return of Message

  • Read Operation
VIDEO_DNR='1'
  • Write Operation
OK: VIDEO_DNR='1'
or
ERROR: VIDEO_DNR='1'
¡@

Applicability

  • PlatfotmT : All Encoders with video processor VX1129

Status

Valid

See Also

VIDEO_BLACKLEVEL, VIDEO_DEINTERLACE, VIDEO_WDR, VIDEO_SKINTONE, V2_SHARPNESS

Example

  • Read Operation
http://ip:port/cgi-bin/encoder?USER=Admin&PWD=123456&VIDEO_DNR
or
http://ip:port/cgi-bin/cmd/VIDEO_DNR
  • Write Operation
http://ip:port/cgi-bin/encoder?USER=Admin&PWD=123456&VIDEO_DNR=1
or
http://ip:port/cgi-bin/cmd/VIDEO_DNR=1
