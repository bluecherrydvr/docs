EVENT_RSPCMD1, EVENT_RSPCMD2, EVENT_RSPCMD3

Description

Get or Set the URL command engine of the event configurations.

When event source is triggered or becomes inactive, the device could send the user defined URL commands to the remote HTTP CGI defined in the HTTP_SERVER.

There are two HTTP server database, named HTTPD1 and HTTPD2, in the device. The notification message, image and video could be uploaded to these servers as events were triggered. The event has to be associated with one of these servers before it was enabled.

Input Argument

        id,OnCmd,OffCmd

id: the method of sending message. It indicates which HTTP server will be the target to sent the URLs. 1: HTTPD1, 2:HTTPD2.

OnCmd: The URL command including CGI path for triggered event. The maximum length is 63.

OffCmd: The URL command including CGI path for event changed to inactive from active state. The maximum length is 63.

If the length of the input argument is longer than its maximum length limitation, the error message will be returned.

Attribute

  • Readable and Writable
  • On-Fly

Return of Message

  • Read Operation

EVENT_RSPCMDn='1,/cgi-bin/MyCgi?NOTIFICATION=Triggered,/cgi-bin/MyCgi?NOTIFICATION=NotTriggered'
or
EVENT_RSPCMDn='1,/cgi-bin/MyCgi?NOTIFICATION=Triggered,'

  • Write Operation

OK: EVENT_RSPCMDn=1,/cgi-bin/MyCgi?NOTIFICATION=Triggered,/cgi-bin/MyCgi?NOTIFICATION=NotTriggered''
or
ERROR: EVENT_RSPCMDn='1,/cgi-bin/MyCgi?NOTIFICATION=Triggered,/cgi-bin/MyCgi?NOTIFICATION=NotTriggered'

where n could be 1, 2 or 3

Applicability

  • EPL Platform : All Devices
  • PlatfotmT : All Devices
  • Multi-Channel Encoder : All Devices

Status

Valid

See Also

EVENT_CONFIG, EVENT_MDn, EVENT_RSPDOn, EVENT_RSPMSGn, EVENT_RSPIMGn, EVENT_RSPGO, HTTP_SERVER

Example

  • read operation
http://ip:port/cgi-bin/encoder?USER=Admin&PWD=123456&EVENT_RSPCMDn
or
http://ip:port/cgi-bin/cmd/encoder?EVENT_RSPCMDn
  • write operation

http://ip:port/cgi-bin/encoder?USER=Admin&PWD=123456&EVENT_RSPCMDn =1,/cgi-bin/MyCgi?NOTIFICATION=Triggered,/cgi-bin/MyCgi?NOTIFICATION=NotTriggered
or
http://ip:port/cgi-bin/cmd/encoder?EVENT_RSPCMDn =1,/cgi-bin/MyCgi?NOTIFICATION=Triggered,/cgi-bin/MyCgi?NOTIFICATION=NotTriggered
where n could be 1, 2 or 3
