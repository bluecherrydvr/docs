NIGHT_GAIN_THD

Description

Get or set the gain threshold for day time state in the camera. The DAY_GAIN_THD and NIGHT_GAIN_THD are the pair of gain thresholds to control the camera to enter the Day&Night state based on lighting condition.

This URL sets the gain level threshold for day/night state change of Day to Night. The higher value in this threshold, more darkness is needed to enter the night mode from day mode.

Some of Day&Night capable camera are able to detect the lighting condition to set it Day&Night state. The source of the Day&Night detection could be CDS, DSP itself or reading the gain level in DSP. Therefore, setting gain thresholds on these detection sources gives the specific lighting condition to enter the day or night state.

To make gain threshold work, the Day&Night Mode has to be in AUTO mode. Refer to VIDEO_DAYNIGHT_MODE URL.

When the CDS is used as the Day&Night detection source, the output of the CDS is treated as the gain level which is used in the DSP. It helps to extend the usage for camera for different Day&Night detection sources.

Some DSP has only one threshold for Day&Night state detection. In this case, this URL is used.

The error message, NIGHT_GAIN_THD not found, will be returned if the encoders and camera do not support Day&Night function or there is no Day&Night detection function.

Input Argument

level
where level is from 0~100.

Attribute

  • Read and Write
  • On-fly

Return of Message

  • Read Operation

NIGHT_GAIN_THD='50'

  • Write Operation

OK: NIGHT_GAIN_THD='50'
or
ERROR: NIGHT_GAIN_THD='50'
or
ERROR: NIGHT_GAIN_THD not found

Applicability

  • EPL Platform : All encoders
  • PlatformT : All encoders

Status

Valid

See Also

VIDEO_DAYNIGHT_MODE, DAY_GAIN_THD, VIDEO_DN_TIMER, VIDEO_DN_TYPE, EXPOSURE_GAIN_GET, Day&Night Control Camera Page Design Specification

Example

  • Read Operation

http://ip:port/cgi-bin/encoder?USER=Admin&PWD=123456&NIGHT_GAIN_THRESHOLD
or
http://ip:port/cgi-bin/cmd/encoder?NIGHT_GAIN_THRESHOLD

  • Write Operation

http://ip:port/cgi-bin/encoder?USER=Admin&PWD=123456&NIGHT_GAIN_THRESHOLD=50
or
http://ip:port/cgi-bin/cmd/encoder?NIGHT_GAIN_THRESHOLD=50
